文内相対位置（語単位）:

mmja H′=.99 unu H′=.93 naugara H′=.90 ca (HS) H′=.11

H′=正規化エントロピー（語単位, MM補正, 1.0=均等分布）。mmja (n=170) は平坦分布。unu (n=40) は文頭偏重。naugara (n=22) は前半にピーク・文末にも分散。ca/HS (n=56) は文末 95% に集中（対比用）。

パッと見ると似た分布をしているにもかかわらず、mmja には unu や naugara にはない位置規則性や談話機能があるという「エセ内省」が、私にはずーっとある（下地 2018）。

しかし「ここで mmja は変ですか?」のような内省判断の引き出しは困難で検証手段がない。

より一般的には、フィールド言語学者が「フィラーっぽいやつ」をすべて DSC FIL と雑にグロスを振って終わりにしがち。区別の妥当性を評価する方法論がない。

提案
gzip圧縮法とサプライザル（情報理論）により、内省に頼らずコーパスから差異を検出することを試みる。

§2 位置規則性

圧縮シャッフル検定 + ブートストラップ信頼区間

各節（EOS/EAC/EQC 区切り, 858節）を語単位に分割。非ターゲット語の POS 列を固定し、TGT_ マーカーの挿入位置のみを一様ランダムに並べ替え 500回 → gzip 圧縮サイズを比較。さらに節をリサンプリングするブートストラップ法（1000回）で Δ/n の 95% 信頼区間を算出。

z = (Corig − μshuf) / σshuf
例: 発話 V mmja N CP V
元データV___TGT_N___CP__V___
shuf 例1V___N___TGT_CP__V___
shuf 例2TGT_N___V___CP__V___

語を入れ替えるのではなく、TGT_ マーカーの位置をランダムに選び直す。非ターゲット語の POS 列は固定。gzip はバイト列の繰り返しパターンを検出するため、TGT_ が特定の POS 文脈に偏って出現するほど圧縮が効く。シャッフルでその噛み合いが壊れると圧縮サイズが増大する。

nzΔ/n95% CI
mmja170−3.09−0.22[−0.21, −0.05]*
unu40−1.88−0.29[−0.40, +0.08]n.s.
naugara22−1.36−0.33[−0.60, +0.04]n.s.

Δ/n = 1語あたり圧縮効果 (bytes)。CI = ブートストラップ 95% 信頼区間 (節リサンプリング ×1000)。

●=Δ/n 点推定、横線=95% CI。ゼロ線を含まない → 有意。

破線=元データ圧縮サイズ。mmja は分布の左端外、unu・naugara は分布内。

解釈
mmja は CI がゼロを含まず、節内の位置に統計的に有意な規則性がある。
unu（n=40）・naugara（n=22）は CI がゼロをまたぎ、検出力不足で判断不能。点推定は規則性方向だが、標本が小さく効果の有無を結論できない。
§5 情報密度

サプライザル （前の語から次の語がどれだけ予測しにくいか）

S(wi) = −log₂ P(wi | wi−1)  S̄corpus = 4.95 bits
実例で理解するサプライザル （= 例文 2, 語単位バイグラム）
バーの高さ = その語のサプライザル（前の語からの予測しにくさ）。mmja 自体は予測しやすい（S=4.43, S̄以下）が、mmja の直後は予測困難（S=8.91）。節境界を挟み新情報（「猫が…」）がここから始まる。

d=Cohen's d（post vs corpus mean）。p=Wilcoxon（Bonferroni 補正済み）。文全体の難しさとは無相関（ρ=.015）。

mmja は低→高密度の遷移点に出現。naugara は逆方向（d=−.12）で非制御的な言い淀みの可能性があるが n=23 で要追加データ。

機能的解釈（暫定。アドバイス求む）

mmja — 談話標識（聞き手への負荷アラート）

直後に予測困難な内容語（N・V・PRN）が来る（post S̄=6.80, d=0.81）。情報密度の遷移点を標示し、「これから難しい内容が来る」と聞き手に予告する。

unu — フィラー（語彙検索？ cf. 日本語について定延・田窪 1995）

後続は名詞に集中（45%）。post S̄=5.68（d=0.32, p<.0001）で平均より有意に高いが mmja ほどではない。名詞句の語彙検索ポーズとして機能か。

定延利之・田窪行則 (1995)「談話における心的操作モニター機構 ―心的操作標識「ええと」と「あの（ー）」―」『言語研究』108: 74-93.

naugara — フィラー（非制御的言い淀み？）

post S̄=4.67（d=−.12, p<.0001）。mmja と逆に予測容易な語が後続するが、効果が小さく、平均に近い。何かを宣告するというより、稀に出てくる言い淀み（後続探索）の可能性あり。ただし、n=23 で要追加データ。自身の S̄ が三者中最高（6.19）なのは低頻度ゆえ。

宮古語伊良部島方言の mmja / unu / naugara は同じ「フィラー」か？

情報理論的指標によるフィラー三形式の機能分化の検証
下地理影 ── 最終レポート

1. 背景と目的

宮古語伊良部島方言には、いずれもフィラー（FIL）とタグされる三つの形式——mmja、unu、naugara——が存在する。既存のコーパスアノテーションでは同一カテゴリに分類されているが、話者の直観としては機能的に異なるという「エセ内省」がある（下地 2018）。

しかし「ここで mmja は変ですか？」のような内省判断の引き出しは困難であり、従来の記述的手法では検証が難しい。そこで本研究では、内省に依存しない情報理論的指標——サプライザルと圧縮シャッフル検定——を用いて、三形式の機能分化を定量的に検証する。

2. データ

伊良部島方言の自然調話コーパス（774発話）を使用。形態素レベルのアノテーションを word_unit_id で語レベルに折り畳み（v4 規則）、節境界（EOS / EAC / EQC / Q）の情報を保持した。

3. 方法

3.1 バイグラム・サプライザル

コーパス全体から add-α smoothing（α = 0.01）付きバイグラムモデルを構築し、各位置のサプライザルを計算した。

S(wi | wi−1) = −log₂ P(wi | wi−1)
P(w | c) = (count(c, w) + α) / (count(c) + αV)

節境界（EOS / EAC / EQC / Q）では文脈を <BOS> にリセットした。ターゲット語の各出現について、直前（pre）・自身（self）・直後（post）のサプライザルを収集し、コーパス平均 S̄ = 4.95 bits との差を Cohen's d および Wilcoxon signed-rank test（Bonferroni k=3 補正）で評価した。

3.2 圧縮シャッフル検定

ターゲット語の文内位置に規則性（パターン）があるかを gzip 圧縮率で検証した。対象形式の文内位置を一様ランダム（uniform）に並べ替え 500 回、gzip 圧縮サイズを比較した。元データが有意に小さければ、位置にパターン（規則性）がある——すなわちターゲット語が特定の POS 文脈に偏って出現していることを意味する。Bonferroni 補正（k=3）適用。

mmja — 談話標識（負荷アラート）

post S̄ = 6.79（d = +0.80, p < .0001）。直後に N (49), V (43), PRN (22) など多岐な内容語が来る。情報密度の低→高遷移点を標示し、「これから難しい内容が来る」と聴き手に予告する。圧縮シャッフル検定でも唯一有意（位置規則性あり）。

unu — フィラー（語彙検索ポーズ）

post S̄ = 5.63（d = +0.29, p < .0001）。平均より有意に高いが mmja ほどではない。後続語は名詞に集中（45%）。圧縮シャッフル検定では有意でなく、位置に規則性が見られない。

naugara — フィラー（アラート解除？）

post S̄ = 4.61（d = −0.15, p < .0001）。mmja とは逆に予測容易な語が後続する。「アラート解除」的な機能の可能性があるが、n = 22 では確定的な解釈は困難。追加データが必要。

mmja のみが有意（z = −5.44, p < .0001）。mmja の文内位置を一様ランダムに並べ替えると圧縮サイズが有意に増大する。すなわち mmja は特定の POS 文脈に偏って出現しており、その配置に規則性がある。unu・naugara は有意でない。

5. 考察

5.1 三形式の機能分化

サプライザル分析と圧縮シャッフル検定の両方から、三形式の機能的差異が確認された。

mmja は単なるフィラーではなく、情報密度の遷移点を標示する談話標識である。直後に予測困難な内容語が来る（d = +0.80）こと、文内位置に規則性がある（圧縮シャッフル検定有意）ことの二点がこれを裏付ける。

unu は後続語が名詞に集中し（45%）、サプライザル上昇も穏やか（d = +0.29）。語彙検索中のポーズとしての古典的フィラー機能に合致する。文内位置の規則性は検出されない。

naugara は mmja と逆方向の効果を示す（d = −0.15）。予測容易な語が後続し、「アラート解除」的な機能の可能性があるが、n = 22 では確定的な解釈は困難であり、追加データが必要。

5.2 方法論的意義

本研究は、内省判断の引き出しが困難な危機言語において、コーパスの数え上げだけで機能の違いを検出できることを示した。サプライザルは話者への問い合わせを一切必要とせず、存在するコーパスだけで計算可能である。これは記述が不十分な他の危機言語にも適用可能な手法である。

6. 結論

宮古語伊良部島方言のフィラー三形式は、情報理論的指標によって明確に分化する。mmja は談話標識として情報密度の遷移点を標示し、unu は語彙検索ポーズとして機能し、naugara はアラート解除の可能性を示すが確定には追加データを要する。「同じフィラー」という従来の分類は再考が必要である。
